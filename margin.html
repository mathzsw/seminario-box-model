Margin

Margin é a propriedade CSS que define o espaço em branco ao redor de um elemento. Ela pode ser usada para criar espaço entre elementos, afastar um elemento de outros ou para centralizar um elemento dentro de seu contêiner.

.center {
margin: auto;
background: lime;
width: 66%;
}
.outside {
margin: 3rem 0 0 -3rem;
background: cyan;
width: 66%;
}
